**How to install:**

Copy the URL to the github:
  When copied open a terminal and run **Git clone "Insert URL Here"**

  After that is completed open Unity and click add: Then select the folder called "The-virtual-Green"

  open the unity project

When you have the project open in Unity:

  **IMPORTANT**

  Go to the project tab and in the assets folder go to Scenes.

  When in scenes drag and drop "Leo's Final Project" into the hieracrchy and remove the scene that was there.

  After that go to file: Build Profiles

  Switch to Meta Quest and make sure the scene you select to run is "Leo's Final Project"

  Press the Build and Run while your output device is set to your meta quest.

  Enjoy the game!
